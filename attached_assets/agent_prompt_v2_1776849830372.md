# تنفيذ 3 ميزات رئيسية — Marketing + Page Builder

## ملخص ما يجب بناؤه

1. **Standalone Leads** — عملاء مستقلون خارج إطار أي حملة
2. **Campaign ROI** — ربحية الحملة (المسجلون × سعر المستوى) − مصاريف الحملة
3. **صفحات هبوط مرتبطة بالحملات** — كل حملة تحصل على صفحة هبوط عامة بنموذج تسجيل
4. **Page Builder محسّن** — تحرير الصفحة الرئيسية بالكامل + بناء صفحات بكتل (sections) + ذكاء اصطناعي

// GET /api/campaigns/:id/roi
router.get("/campaigns/:id/roi", requireAuth, async (req: Request, res: Response) => {
  const user = (req as any).user;
  if (!["admin", "accountant"].includes(user.role)) return res.status(403).json({ error: "Forbidden" });

  const campaignId = parseInt(req.params.id);
  if (isNaN(campaignId)) return res.status(400).json({ error: "Invalid id" });

  // جلب العملاء المسجلين
  const registeredLeads = await db
    .select({ id: leadsTable.id, childName: leadsTable.childName, preferredLevel: leadsTable.preferredLevel })
    .from(leadsTable)
    .where(and(eq(leadsTable.campaignId, campaignId), eq(leadsTable.status, "registered")));

  // جلب مصاريف الحملة
  const expenses = await db
    .select()
    .from(campaignExpensesTable)
    .where(eq(campaignExpensesTable.campaignId, campaignId));

  // جلب المستويات مع أسعارها
  const levels = await db.select({ id: levelsTable.id, name: levelsTable.name, monthlyFee: levelsTable.monthlyFee }).from(levelsTable);

  const totalExpenses = expenses.reduce((s, e) => s + e.amount, 0);

  res.json({
    registeredLeads,
    expenses,
    levels,
    totalExpenses,
    registeredCount: registeredLeads.length,
  });
});

// POST /api/campaigns/:id/expenses
router.post("/campaigns/:id/expenses", requireAuth, async (req: Request, res: Response) => {
  const user = (req as any).user;
  if (!["admin", "accountant"].includes(user.role)) return res.status(403).json({ error: "Forbidden" });

  const campaignId = parseInt(req.params.id);
  const { description, amount, category } = req.body;
  if (!description || !amount) return res.status(400).json({ error: "description and amount required" });

  const [expense] = await db.insert(campaignExpensesTable).values({
    campaignId,
    description: description.trim(),
    amount: parseFloat(amount),
    category: category ?? "other",
  }).returning();
  res.status(201).json(expense);
});

// DELETE /api/campaigns/expenses/:id
router.delete("/campaigns/expenses/:id", requireAuth, async (req: Request, res: Response) => {
  const user = (req as any).user;
  if (!["admin", "accountant"].includes(user.role)) return res.status(403).json({ error: "Forbidden" });
  const id = parseInt(req.params.id);
  await db.delete(campaignExpensesTable).where(eq(campaignExpensesTable.id, id));
  res.json({ message: "Deleted" });
});

## PART D — Page Builder المحسّن بالذكاء الاصطناعي

### D1 — إعادة بناء صفحة Web Content

استبدل `artifacts/kidspeak/src/pages/admin/web-content/index.tsx` بالكامل بـ implementation جديد يحتوي على:

**Tabs:**
1. **الصفحة الرئيسية** — نفس ما هو موجود (Hero + Testimonials + Open Day) مع تحسينات
2. **الصفحات المخصصة** — قائمة الصفحات مع زر "تعديل" يفتح Page Builder
3. **إنشاء بالذكاء الاصطناعي** — tab جديد

**Page Builder محسّن (Modal كامل الشاشة):**

بدلاً من Textarea بسيط، أنشئ builder بـ Sections/Blocks:

```
الأقسام المتاحة للإضافة:
- Hero (عنوان كبير + وصف + زر CTA)
- Features (3 ميزات مع أيقونات)
- Text Block (نص عادي)
- Image + Text
- CTA Section (دعوة للتسجيل)
- Registration Form (يربط بـ campaign)
```

كل section له:
- زر تحرير محتواه
- زر حذف
- زر تحريك لأعلى/أسفل

**Schema الجديد للصفحات** — غيّر contentEn/contentAr من HTML خام إلى JSON:
```json
{
  "sections": [
    { "id": "1", "type": "hero", "data": { "titleEn": "...", "titleAr": "...", "subtitleEn": "...", "ctaText": "...", "ctaLink": "..." } },
    { "id": "2", "type": "features", "data": { "items": [{ "icon": "🎯", "titleEn": "...", "descEn": "..." }] } },
    { "id": "3", "type": "text", "data": { "contentEn": "<p>...</p>", "contentAr": "<p>...</p>" } }
  ]
}
```

**عارض الصفحة العامة** (`public-page/index.tsx`) يُعدَّل ليقرأ هذا JSON ويعرض كل section بشكله المناسب.

**لا تنسَ**: الصفحات القديمة التي contentEn/contentAr هي HTML خام يجب تحويلها تلقائياً:
```typescript
function parseContent(content: string) {
  try { return JSON.parse(content); } catch { return { sections: [{ id: "1", type: "text", data: { contentEn: content, contentAr: content } }] }; }
}
```

      <Input type="password" value={apiKey} onChange={e => setApiKey(e.target.value)} placeholder="sk-..." dir="ltr" />
      <Button onClick={saveKey}>Save Key</Button>
    </div>
  );
}

// مولّد الصفحة
function AIPageGenerator({ isRTL }: { isRTL: boolean }) {
  const [prompt, setPrompt] = useState("");
  const [generating, setGenerating] = useState(false);
  const [result, setResult] = useState<any>(null);

  const generate = async () => {
    const apiKey = localStorage.getItem("ai_api_key");
    const provider = localStorage.getItem("ai_provider") ?? "anthropic";
    if (!apiKey || !prompt) return;
    
    setGenerating(true);
    try {
      let sections;
      
      if (provider === "anthropic") {
        const response = await fetch("https://api.anthropic.com/v1/messages", {
          method: "POST",
          headers: {
            "Content-Type": "application/json",
            "x-api-key": apiKey,
            "anthropic-version": "2023-06-01",
          },
          body: JSON.stringify({
            model: "claude-opus-4-6",
            max_tokens: 2000,
            messages: [{
              role: "user",
              content: `Create a landing page structure in JSON for: "${prompt}"
              
Return ONLY valid JSON with this structure (no markdown, no explanation):
{
  "titleEn": "page title in English",
  "titleAr": "page title in Arabic",
  "sections": [
    { "id": "1", "type": "hero", "data": { "titleEn": "...", "titleAr": "...", "subtitleEn": "...", "subtitleAr": "...", "ctaTextEn": "Register Now", "ctaTextAr": "سجّل الآن" } },
    { "id": "2", "type": "features", "data": { "items": [{ "icon": "🎯", "titleEn": "...", "titleAr": "...", "descEn": "...", "descAr": "..." }, { "icon": "⭐", "titleEn": "...", "titleAr": "...", "descEn": "...", "descAr": "..." }, { "icon": "🏆", "titleEn": "...", "titleAr": "...", "descEn": "...", "descAr": "..." }] } },
    { "id": "3", "type": "text", "data": { "contentEn": "<p>detailed description...</p>", "contentAr": "<p>وصف تفصيلي...</p>" } },
    { "id": "4", "type": "cta", "data": { "titleEn": "...", "titleAr": "...", "buttonTextEn": "Register Now", "buttonTextAr": "سجّل الآن" } }
  ]
}`,
            }],
          }),
        });
        const data = await response.json();
        const text = data.content?.[0]?.text ?? "";
        sections = JSON.parse(text.replace(/```json|```/g, "").trim());
      }
      
      setResult(sections);
    } catch (e) {
      toast({ title: "Failed to generate page", variant: "destructive" });
    } finally {
      setGenerating(false);
    }
  };

  // عرض النتيجة مع زر "إنشاء صفحة" الذي يأخذ result ويفتح Page Builder
  return (
    <div className="space-y-4">
      <AIPageGeneratorSettings isRTL={isRTL} />
      <Textarea 
        value={prompt} 
        onChange={e => setPrompt(e.target.value)}
        placeholder={isRTL ? "صف الصفحة التي تريد إنشاؤها... مثال: صفحة هبوط ليوم مفتوح لمدرسة لغة إنجليزية" : "Describe the page you want to create..."}
        rows={3}
      />
      <Button onClick={generate} disabled={generating || !prompt} style={{ backgroundColor: "#1B2E8F", color: "white" }}>
        {generating ? (isRTL ? "جارٍ الإنشاء..." : "Generating...") : (isRTL ? "✨ إنشاء الصفحة" : "✨ Generate Page")}
      </Button>
      
      {result && (
        <div className="p-4 rounded-xl border bg-green-50">
          <p className="text-sm font-semibold text-green-700 mb-2">{isRTL ? "تم إنشاء الصفحة!" : "Page generated!"}</p>
          <p className="text-xs text-slate-500">{result.sections?.length ?? 0} {isRTL ? "أقسام" : "sections"}</p>
          <Button onClick={() => {/* فتح Page Builder مع هذه الـ sections */}} className="mt-2" size="sm" style={{ backgroundColor: "#1B2E8F", color: "white" }}>
            {isRTL ? "فتح في المحرر" : "Open in Editor"}
          </Button>
        </div>
      )}
    </div>
  );
}
